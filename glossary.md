## Операционные термины

**Рейс** — операционный цикл PÄSH: закупка на Алтын-Орде → развоз по ПВЗ →
выдача заказов покупателям. В альфа-периоде: вторник и пятница.
Стоимость рейса: своя машина 5 000₸ бензин, газель 13 000₸ + 3 000₸/доп. точку.

**ПВЗ** (Пункт Выдачи Заказов) — стеллаж в магазине-партнёре без собственного
овощного отдела. Аренда: 50 000₸/мес фиксированно. Покупатель забирает заказ
по QR-коду до 22:00 дня доставки.

**Правило 22 часов** — заказ необходимо забрать до 22:00 дня доставки.
Невостребованные заказы = убыток. Отслеживает агент М.

**Алтын-Орда** — крупнейший оптовый рынок Центральной Азии, источник закупки PÄSH.
Грузчики-тележечники: 3 000₸/рейс. Оплата поставщикам: Kaspi QR.

**Манифест** — список заказов для рейса: что купить, в каком количестве, для какого ПВЗ.
Формирует агент М, исполняет 007.

**Предзаказ** — принцип нулевых остатков: сначала собираем заказы, потом едем на базар.
Покупаем ровно под спрос. Деньги поступают ДО закупки (cash-upfront цикл).

## Ассортиментные термины

**Крючок** — SKU с наценкой 15–20%, цель — привлечение трафика и формирование привычки.
Примеры: огурцы (500₸/кг), бананы (787₸/кг), помидоры (625₸/кг).
Покупатель оценивает PÄSH по цене крючков → доверие переносится на маржу.

**Маржинальный продукт (маржа)** — SKU с наценкой 50–60%, цель — прибыль.
Примеры: авокадо (1 100₸/шт), ананас (1 875₸/шт), гранат (1 250₸/кг).
Правило: маржинальный продукт всегда дешевле магазина в 1.5–2 раза.

**Подавленный спрос** — спрос, который существует но не реализуется из-за высокой цены.
Классика: гранат по 3 053₸ — не берут. По 1 250₸ — берут все.
PÄSH не отбирает долю у магазина — создаёт новый спрос.

## Финансовые термины

**Средняя наценка** — минимум 35% для покрытия скрытых расходов и прибыли.
Структура: крючки 15–20%, сезонные 30%, цитрусы 35%, премиум 50–60%.

**Скрытые расходы** — ~10.5% от выручки, суммарно:
- Эквайринг TipTop Pay (БЦК): 3%
- Налог ИП упрощёнка: 3%
- Обналичка для базара: 1.5%
- Брак и потери: 3%
→ Итог: наценка < 35% = убыток даже при нулевых операционных расходах.

**Безубыток** — 28 заказов/рейс при 2 ПВЗ, AOV 7 000₸, наценка 35%.
Ниже 28 заказов = рейс убыточен.

**AOV** (Average Order Value, Средний чек) — ключевая метрика.
Целевой: > 7 000₸. Текущий альфа: отслеживается в Supabase.

**Retention** — доля покупателей, вернувшихся на второй рейс.
Порог для инвестора (pre-seed): > 50%.
Главный долгосрочный актив в продуктовом ритейле — привычка.

**Cash-upfront цикл** — оборотный капитал практически нулевой:
покупатель платит через Kaspi QR → деньги поступают до закупки →
поставщик получает Kaspi QR → ноль кредитного риска.

**Unit-экономика** — экономика одного рейса. До доказательства unit-экономики — не идём к инвестору.

## Командные термины (007-архитектура)

**Мани-Пенни 💃** — AI-агент клиентского фронта.
WA SIM#1 + TG @pash_client_price_bot. Принимает заказы, отвечает на вопросы.
Layer 1→2 эволюция.

**М 🎩** — AI-агент бэкофиса: ценообразование, манифесты, возвраты, дайджесты.
WA SIM#2 + TG internal. Командный центр. Robin = М.

**Q 🔬** — AI-агент Vision-контроля качества и еженедельной аналитики.
Проверяет стеллажи по фото, следит за ценовым мониторингом.

**007 🔫** — Закупщик в поле (человек). Едет на Алтын-Орду по манифесту от М.

**009 🚚** — Водитель-экспедитор (человек). Везёт товар по ПВЗ.

**@pash_club** — TG-группа первых покупателей: «PÄSH · Первые».

**@pash_channel** — Публичный TG-канал PÄSH.

## Стратегические термины

**3 слоя / 3 qabat:**
- Layer 1: PÄSH Ритейл (сейчас) — доказывает модель
- Layer 2: PÄSH OS (2027) — AI-бэкофис для phygital бизнесов СНГ
- Layer 3: Platform/Jobs of AI (2028) — сетевой эффект, Ташкент, Бишкек, Тбилиси

**Оцифрованный базар** — ключевое позиционирование. PÄSH ≠ западный e-grocery.
Ocado = 3 огурца в упаковке для британцев. PÄSH = ящик с Алтын-Орды у дома.

**Wrapper over bazaar economy** — PÄSH добавляет цифровой слой над инфраструктурой
Алтын-Орды не заменяя её. Аналогия: Perplexity над GPT.

**PÄSH OS** — Layer 2: AI-бэкофис как SaaS для мясников, пекарен, молочников,
цветочников с 2–15 заказами в день через WhatsApp.

**Эффект Мамдани** — визуальный приём бренда: жирный янтарный текст (#FFAB05)
с красным смещением (#E81806), offset 60–80px, blur 0.

**Birge arzan** — главный слоган на общетюркской латинице: «Вместе дешевле».

## Технические термины

**Supabase** — основная БД.
Проект: `nzskcvwghqadvobjrppf.supabase.co`.
Таблицы: products, price_snapshots, alerts, orders, micro_districts, users, price_queries.
RLS включён. Читать: anon key. Писать: service role key (только бэкенд).

**TipTop Pay** — платёжный шлюз (БЦК), эквайринг 3%.
Webhook → Railway → Supabase (обновление статуса заказа).

**Kaspi QR** — платёжный метод поставщиков на Алтын-Орде. Критически важен для
cash-upfront цикла.

**api.pash.kz** — основное API. Документация: запросить у Amir.

**Railway** — хостинг Python-ботов (Мани-Пенни, М, Q).

**Vercel** — деплой HTML-дашбордов.

**pash-price-bot** — внутренний бот мониторинга цен (Python + Gemini Vision).

**pash-client-price-bot** — публичный бот (@pash_client_price_bot), краудсорсинг цен.
