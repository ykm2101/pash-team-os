# PÄSH · Бизнес-модель
> Tier: stable. Изменения только через решение Erdos.

## Главная идея

PÄSH — оцифрованный базар, не западный e-grocery.
Ocado = 3 огурца в упаковке для британцев. PÄSH = ящик с Алтын-Орды у твоего дома.
Казахстан перепрыгнул 15 лет эволюции e-grocery через Kaspi. Окно открыто.

## Модель одной строкой

Предзаказ → закупка на Алтын-Орде ровно под спрос → развоз в ПВЗ-стеллажи → самовывоз по QR до 22:00.

## Asset-light: что НЕТ

- ❌ Нет склада (ПВЗ в партнёрских магазинах, 50 000₸/мес аренда)
- ❌ Нет курьеров (самовывоз)
- ❌ Нет холодильников (свежий завоз ежедневно)
- ✅ Cash-upfront: деньги поступают ДО закупки

## 3 слоя (3 qabat)

**Layer 1: PÄSH Ритейл (сейчас, май 2026)**
Ящики с Алтын-Орды → ПВЗ → покупатель. Доказывает модель. Себя кормит со 2-го месяца.

**Layer 2: PÄSH OS (2027, 20+ точек)**
AI-агенты управляют закупкой, ценами, диспетчеризацией, Vision-контролем.
1 оператор = 50+ точек. Расходы не растут линейно.

**Layer 3: Platform для СНГ (2028)**
PÄSH OS как SaaS для других phygital бизнесов: мясники, пекарни, молочники, цветочники.
Ташкент, Бишкек, Тбилиси. SaaS-мультипликатор + сетевой эффект.

## Подавленный спрос — ключевой инсайт

Гранат по 3 053₸ — не берут. По 1 250₸ — берут все.
PÄSH не отбирает долю у магазина — **создаёт новый спрос из воздуха.**

Когда цена падает до доступного уровня → человек покупает впервые → нравится →
формируется привычка. **Привычка — главный долгосрочный актив.**

## Дифференцированная маржа

| Группа | Наценка | Роль |
|--------|---------|------|
| Базовые (огурцы, бананы) | 15–20% | Крючок — трафик и привычка |
| Сезонные (мандарин, гранат) | 30% | Объём |
| Цитрусы (апельсин, грейпфрут) | 35% | Стабильный спрос |
| Премиум (авокадо, ананас) | 50–60% | Прибыль |

Психология: покупатель оценивает PÄSH по цене огурцов → доверие переносится на авокадо.

## Ров

1. Личные отношения с оптовиками на Алтын-Орде (Магнум физически не может)
2. Asset-light — новую точку открываем за 2 дня
3. AI-бэкофис невидим снаружи — копируют витрину за месяц, операционку за год
4. 63 личных договорённости с точками — кто занял, тот победил
5. Клуб покупателей — ощущение инсайдера, Магнум слишком безликий

## Конкурентная среда

- **Kaspi** — монополия, не пускает. Одновременно угроза и потенциальный партнёр/покупатель.
- **Магнум** — основной ценовой ориентир (авокадо 1 630₸ vs PÄSH 1 100₸).
- **Арбуз.kz** — ближайший прямой конкурент.
- **Ocado** — неправильная модель для рынка (британская фасовка).
